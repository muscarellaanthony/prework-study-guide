HTML

  • The head element contains information about the webpaeg.
  • The body element represents the visible content shown to the user.
  • Most of the writing for a webpage will go into to body.
  • Headers should be used in order from 1 to 6

CSS

  • A margin indicaetes how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Always class names in the css sheet with a period.
  • Use an asterisk to apply declarations to all elements on a page.
  • Use several selectors separated by commas to apply declarations to all selectors.

Git

  • git staus: shows which branch you are on.
  • git checkout -b branch-name: creates a new branch and swithces to it.
  • git add - A: lets git know to stage all files.
  • always stage files after saving any changes
  • git commit -m "note": commits change and adds a note.
  • git pull origin main: pulls source code from github.
  • always pull source code before pushing changes.
  • git push origin "branch name with angled brackets": push bracnh to your branch

JavaScript

  • Declare varibales using let or const.
  • Let is changeable through the code, const is not.
  • Strict equality: ===
  • Not: ! Does not equal: !==
  • Null: no object undefine: no value
  • console.log prints a message in the console log.
